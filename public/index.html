Welcome to the Guild Roster API

This homepage will serve as the official source of documentation for topics relating to this service.

What it is

We created this as a side project in order to create a tool that could help guilds visualize their current roster and examine the skillsets and other various information about each of their guildmembers. Currently, the project supports a simple roster upkeep system, but we have plans to continue working on the project over time and add features such as:

  • Announcements Page
  • Raiding Calendar
  • Social Event Calendar
  • General Forums
  • Member Application Form
  • Guild Highlights
  • And more...

If you are interested in having any of these services established for your guild, send us your email!
